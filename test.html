Fee-Nature

Fee-Nature, un projet site statique, elle voulait mettre en valeur ses œuvres, cette artiste créer des toiles et des tableaux avec des éléments naturel. Elle voulait un site web simple et facile à naviguer, avec une page contact. Le code est en Html 5 / Css 3 avec comme serveur github pour commencer il sera sur le cms Bold ou Wordpress et sur un serveur priver.
